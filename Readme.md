## 📖 About This Challenge

This repository documents my **101-day learning journey**, where each day I create a unique, fun mini-project to strengthen my web development skills. The goal is to learn something new every single day while building a diverse portfolio of interactive web experiences.

### 🎯 Objectives
- ✅ Master **HTML, CSS, and JavaScript** fundamentals
- ✅ Explore advanced CSS concepts (animations, transitions, effects)
- ✅ Build interactive UI components
- ✅ Practice responsive design and user experience
- ✅ Create a portfolio of creative projects
- ✅ Build consistent coding habits

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Text editor (VS Code recommended)
- Git installed on your machine

### Installation

```bash
# Clone the repository
git clone https://github.com/karimiwambui383/101-Fun-Projects-in-101-Days.git

# Navigate to the project directory
cd 101-Fun-Projects-in-101-Days

# Open any project folder and run the HTML file
open "Day 1/index.html"
```

### Running a Project

Simply navigate to any day's folder and open the `index.html` file in your browser. Each project is self-contained with no dependencies.

```bash
# Example: View Day 1 project
cd "Day 1"
# Open index.html in your browser or run a local server
```
